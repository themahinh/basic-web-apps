# Basic Web Apps

This repo documents my journey of learning web development by building small, practical apps in public.

## Projects

- [Weather App](./projects/weather-app/) — A simple app that shows weather data using an API.
- [Random Quotes Generator](./projects/quotes-generator/) — Get random quotes from an API.
- [To-Do List](./projects/todo-app/) — A basic task manager with local storage.
- More projects coming soon.

## About

Each folder contains a small beginner-friendly project built to practice core frontend skills like:
- HTML structure
- CSS styling
- JavaScript logic
- DOM manipulation
- API integration

## Tech Stack

- HTML
- CSS
- JavaScript
- REST APIs
- Local Storage

## Why This Repo

This repo is mainly for learning in public.  
I use it to build small apps, test ideas, and track my progress as I grow in web development.
